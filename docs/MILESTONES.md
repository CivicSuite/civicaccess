Version sync, release artifacts, GitHub release, persistence, and CivicCore dependency alignment.

Accessible forms, accessible publishing workflows, WCAG review support, plain-language rewrites, multilingual draft variants, ADA Title II review support, tagged-PDF expectations, records-ready exports, browser QA, adversarial tests, and release artifacts.
